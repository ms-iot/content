Saját eszközeinek internetes hálózata

Az eszközök internetes hálózata (Internet of Things, IoT) összekapcsolja egymással az eszközöket, a érzékelőket, a felhőt, az adatokat – és velük az Ön fantáziáját.

Kezdje el most!
Az újdonságok áttekintése: Mostantól Raspberry Pi 3-ason is
Néhány szó a Windows 10 IoT Core-ról

Ismerje meg, milyen funkciókat kínál a Windows 10 IoT Core: a Windows kényelmét egyesíti az IoT erejével.

Tudnivalók a kezdőcsomagról

Az Adafruit készített egy kezdőcsomagot a Windows 10 IoT Core-hoz. Ebben megtalálhatók azok a kompatibilis összetevők, érzékelők és alkatrészek, amelyekből Ön izgalmas újdonságokat építhet.

Építsen saját Windows IoT-eszközt

Gyorsan készíthet prototípusokat különféle Windows 10 IoT Core rendszerű eszközökre szánt Windows IoT-megoldásokhoz. A Windows 10 sokoldalú eszközöket biztosít, amelyekkel gyors a fejlesztés és az eszközökre való telepítés.

Csatlakozzon a Maker közösséghez

Itt kapcsolatot teremthet a Maker közösség többi tagjával, és megoszthatja saját kódjait és egyéb munkáit a GitHubon. A közösséghez csatlakozva láthatja, min dolgoznak mások, és Ön is beleszólhat az SDK későbbi kiadásainak tartalmába.

Szerezze be az új Raspberry Pi 3-as lapot Windows 10 IoT Core rendszerrel, amely már a NOOBS telepítő része. A Windows Insider programban kiadtuk a Windows 10 IoT Core újabb Insider Preview előzetesét, amely teljes mértékben képes kihasználni a Raspberry Pi 3 lehetőségeit.
