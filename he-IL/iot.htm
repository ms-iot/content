האינטרנט של הדברים שלך

האינטרנט של הדברים (IoT) מאחד התקנים, חיישנים, ענן, נתונים ואת הדמיון שלך.

התחל בעבודה עכשיו
גלה מה חדש: כעת ב- Raspberry Pi 3
מהו Windows 10 IoT Core?

גלה את התכונות ואת הפונקציונליות שמספק Windows 10 IoT Core. זוהי הקלות של Windows בשילוב עם העוצמה של IoT.

למד אודות ה-Starter Pack

Adafruit יצרה Starter Pack עבור Windows 10 IoT Core. החבילה כוללת רכיבים, חיישנים וחלקים תואמים, שאפשר להשתמש בהם כדי להתחיל ליצור פרויקטים מלהיבים.

בנה את התקן Windows IoT שלך

תכנן במהירות אב-טיפוס ובנה את פתרונות Windows IoT במגוון התקנים שבהם פועל Windows 10 IoT Core. Windows 10 מעניק לך כלים רבי-עוצמה שמאפשרים לך לפתח במהירות ולבצע פריסה להתקן שלך.

הצטרף אל קהילת המייקרים

צור קשר עם מייקרים אחרים כדי לשתף קוד ולתרום דרך GitHub. הצטרף לקהילה כדי לראות מה אחרים בונים ותוכל להשפיע על גירסאות עתידיות של ה-SDK.

פרוייקטים מומלצים

הצג הכל
בקרה במעגל סגור, חיישנים מרוחקים ו- UX מרוחק ב- Raspberry Pi 3
מאת Windows IoT

השתמש בבקר PID כדי לנטר את המהירות של גלגל ולהגיע בעקביות לסל“ד הרצוי באמצעות Raspberry Pi 3 החדש.

מפה עולמית של מייקרים
מאת Windows IoT

צור אפליקציה פשוטה המחוברת לאינטרנט כדי לבדוק את המצב של API אינטרנט והשתמש במידע הזה כדי להפעיל נורת LED.

תחנה מטאורולוגית המשתמשת ב- Windows 10
מאת Windows IoT

צור תחנה מטאורולוגית שמשתמשת ב-API החדשים של התקנים ב-Windows 10 במטרה לנצל את העוצמה של Windows תוך קיום אינטראקציה עם העולם האמיתי דרך חיישנים ומפעילים.

ערכת רובוטים
מאת Windows IoT

בנה רובוט עם גלגלים באמצעות ערכת רובוטים, ולאחר מכן חבר מוט היגוי מסוג USB כדי לשלוט בתנועה שלו באמצעות לוח Raspberry Pi 2 או 3 שפועל בו Windows.

Raspberry Pi 3 זמין כעת

התנסה בלוח Raspberry Pi 3 החדש עם Windows 10 IoT Core, כעת חלק ממתקין NOOBS. פרסמנו גירסה חדשה של Windows 10 IoT Core Insider Preview דרך התוכנית למשתתפי Windows Insider, שתאפשר לך לנצל את מלוא היתרונות של Raspberry Pi 3.

  • רכוש את Raspberry Pi3
  • הפוך למשתתף Windows Insider
  • התחל בעבודה עם NOOBS
