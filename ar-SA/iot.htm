إنترنت أشيائك

تجمع إنترنت الأشياء (IoT) ما بين الأجهزة وأدوات الاستشعار والسحابة والبيانات وخيالك.

بدء الاستخدام الآن
تعرّف على الجديد: الآن على Raspberry Pi 3
ما هو Windows 10 IoT Core؟

اكتشف الميزات والوظائف التي يوفرها Windows 10 IoT Core. إنه يجمع سهولة Windows جنبًا إلى جنب مع قوة إنترنت الأشياء (IoT).

التعرف على Starter Pack

أنشأ أدافرويت Starter Pack لـ Windows 10 IoT Core. تحتوي الحزمة على مكونات وأدوات استشعار وأجزاء متوافقة يمكنك استخدامها لبدء إنشاء مشروعات مثيرة.

قم بإنشاء جهاز Windows IoT الخاص بك

قم بإنشاء النموذج الأولي لحلول Windows IoT الخاصة بك بسرعة على مجموعة متنوعة من الأجهزة التي تعمل بنظام تشغيل Windows 10 IoT Core. يوفر لك Windows 10 أدوات قوية تتيح لك تطويرها بشكل سريع ونشرها إلى جهازك.

الانضمام إلى مجتمع المصنعين

تواصل مع مصنعين آخرين لمشاركة التعليمات البرمجية وتقديم مساهمات من خلال GitHub. انضم إلى المجتمع لمعرفة ما يصممه الآخرون وهو ما سيكون له تأثيره على الإصدارات المستقبلية من SDK.

المشروعات المميزة

عرض الكل
عنصر تحكم الحلقة المغلقة وأجهزة الاستشعار البعيدة وتجربة المستخدم البعيد على Raspberry Pi 3
بواسطة Windows IoT

استخدم جهاز تحكم PID لمراقبة سرعة العجلة وتحقيق الوصول الثابت إلى معدل الدورة في الدقيقة المطلوب على لوحة Raspberry Pi 3 الجديدة.

خريطة الصانعين على مستوى العالم
بواسطة Windows IoT

قم بإنشاء تطبيق بسيط متصل بالإنترنت للاستعلام عن حالة واجهة برمجة التطبيقات (API) للويب واستخدم تلك المعلومات لتبديل LED.

محطة الطقس باستخدام Windows 10
بواسطة Windows IoT

قم بإنشاء محطة طقس تستخدم واجهات برمجة تطبيقات أجهزة جديدة في نظام Windows 10 للاستفادة من قدرة Windows أثناء التفاعل مع العالم الحقيقي عبر أدوات الاستشعار ووحدات التشغيل.

مجموعة أدوات الروبوت
بواسطة Windows IoT

قم بإنشاء روبوت بعجلات باستخدام مجموعة أدوات الروبوت، وقم بتوصيل ذراع لعب USB للتحكم في حركته باستخدام لوحة Raspberry Pi 2 أو 3 التي تعمل بنظام التشغيل Windows.

لوحة Raspberry Pi 3 متوفرة الآن

احصل على لوحة Raspberry Pi 3 الجديدة بنظام Windows 10 IoT Core، الذي يشكّل الآن جزءًا من مثبت NOOBS. لقد أطلقنا إصدارًا جديدًا من Windows 10 IoT Core Insider Preview من خلال برنامج Windows Insider من شأنه أن يسمح لك بالاستفادة إلى أقصى حد من لوحة Raspberry Pi 3.

  • اشترِ لوحة Raspberry Pi3
  • شارك في برنامج Windows Insider
  • بدء استخدام NOOBS
