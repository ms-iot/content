¿Qué es Windows 10 IoT Core?

Descubre las características y funciones que ofrece Windows 10 IoT Core. La comodidad de Windows combinada con la potencia de IoT.

Obtener información sobre el paquete de inicio

Adafruit ha creado un paquete de inicio para Windows 10 IoT Core. El paquete incluye componentes compatibles, sensores y elementos que puedes usar para empezar a crear proyectos fascinantes.

Crear tu dispositivo de Windows IoT

Diseña prototipos rápidamente y crea soluciones de Windows IoT en una variedad de dispositivos que ejecutan Windows 10 IoT Core. Windows 10 ofrece herramientas eficaces que te permiten desarrollar e implementar rápidamente en el dispositivo.

Unirse a la comunidad de fabricantes

Conecta con otros fabricantes para compartir código y hacer contribuciones a través de GitHub. Únete a la comunidad para ver qué están creando otros desarrolladores y para influir en futuras versiones del SDK.
