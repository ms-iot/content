您的物聯網

物聯網 (IoT) 能串聯您的裝置、感應器、雲端、資料與您的想像。

立即開始使用
查看新功能： Raspberry Pi 3 的新功能
Windows 10 IoT 核心版是什麼?

探索 Windows 10 IoT 核心版提供的功能。它結合了 Windows 的簡易與 IoT 的性能。

了解 Starter Pack

Adafruit 已經為 Windows 10 IoT 核心版建立了 Starter Pack。此套件包括您可以用來開始建置有趣專案的相容元件、感應器與組件。

建置您的 Windows IoT 裝置

在多種執行 Windows 10 IoT 核心版的裝置上，快速建立您的 Windows IoT 解決方案原型並加以建置。Windows 10 提供您可用來快速開發並部署到您裝置的強大工具。

加入 Maker 社群

透過 GitHub 與其他 Maker 聯繫，以分享程式碼並做出貢獻。加入社群即可查看其他人建置的項目，以及對未來發佈的 SDK 產生影響。
