Get your imaging tools

For the next two steps, we'll be focusing on creating images. There are two types of images that you can build: test images, which are ideal for quickly accessing and modifying devices, and retail images, which are more secure and better designed for manufacturing. For now, we'll be building test images, but resources for customizations and more can be found in our IoT Core Manufacturing Guide.

Devices and storage

  • A PC running the latest available version of Windows 10 (downloadable here) with at least 15 GB of free space
  • The development platform with the silicon you have chosen
  • Setup with an attached display

Software

  • Windows Assessment and Deployment Kit
  • IoT Core .iso package
  • IoT Core ADK Add-Ons
