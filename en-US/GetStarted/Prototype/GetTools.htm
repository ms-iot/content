Get your developer tools

Now that you've installed IoT Core on your device, we'll need to download the tool that you'll use to write your first application.

Download and install the appropriate version of Visual Studio for your work.

Visual Studio Downloads

Download Visual Studio Community 2017 (For most developers)

Download Visual Studio Enterprise 2017 (For enterprise developers)

Download Visual Studio Professional 2017 (For individuals and small teams)

Note: If you want to use C++ for your projects, make sure to check the UWP C++ checkbox upon downloading.

Visual Studio allows developers to deploy and debug applications remotely. With the power of Windows IoT and Visual Studio, UWP can be used to write applications that run on all Windows device in any language.
