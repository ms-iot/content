Get your developer tools

Now that you've installed Windows 10 IoT Core on your device, you'll need to download the version of Visual Studio that works best for you in order to write your first application.

Visual Studio Downloads

Download Visual Studio Community 2017 (For most developers)

Download Visual Studio Enterprise 2017 (For enterprise developers)

Download Visual Studio Professional 2017 (For individuals and small teams)

Note: If you want to use C++ for your projects, make sure to check the UWP C++ checkbox when downloading.

Visual Studio allows developers to deploy and debug applications remotely. With the power of Windows IoT and Visual Studio, the Universal Windows Platform (UWP) can be used to write applications that run on all Windows device in any language.
