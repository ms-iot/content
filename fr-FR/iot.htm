L’Internet de vos objets

L’IoT (Internet of Things) permet de regrouper des appareils, des capteurs, des clouds, des données... au fil de votre imagination.

Prise en main
Découvrir les nouveautés : dès maintenant sur Raspberry Pi 3
Qu’est-ce que Windows 10 IoT Standard ?

Découvrez les fonctionnalités offertes par Windows 10 IoT Standard. Cette solution combine la simplicité d’utilisation de Windows avec la puissance d’IoT.

En savoir plus sur le pack de démarrage

Adafruit a créé un pack de démarrage pour Windows 10 IoT Standard. Ce pack contient des composants, capteurs et pièces compatibles que vous pouvez utiliser pour commencer à concevoir des projets innovants.

Créer votre appareil Windows IoT

Réalisez un prototype et créez rapidement vos solutions Windows IoT sur de nombreux appareils exécutant Windows 10 IoT Standard. Windows 10 vous offre de puissants outils pour développer rapidement des applications et les déployer sur votre appareil.

Rejoindre la communauté de créateurs

Contactez d’autres créateurs afin de partager du code et d’apporter votre contribution par le biais de GitHub. Rejoignez la communauté pour voir les créations d’autres personnes et influer sur les versions à venir du SDK.

Projets proposés

Afficher tout
Contrôle en circuit fermé, capteurs et expérience utilisateur à distance sur Raspberry Pi 3
Par Windows IoT

Utilisez un contrôleur PID sur le nouveau Raspberry Pi 3 pour surveiller la vitesse d’une roue et faire en sorte d’atteindre le nombre de tours par minute souhaité.

Carte du monde des créateurs
Par Windows IoT

Créez une application simple connectée à Internet pour interroger le statut d’une API web et utiliser ces informations pour activer/désactiver une LED.

Station météo sur Windows 10
Par Windows IoT

Créez une station météo utilisant les nouvelles API d’appareil Windows 10 pour exploiter la puissance de Windows tout en interagissant avec le monde réel à travers des capteurs et des positionneurs.

Kit de création de robot
Par Windows IoT

Construisez un robot à roulettes à l’aide d’un kit de création de robot, puis connectez une manette de jeu USB pour contrôler ses mouvements avec votre carte Raspberry Pi 2 ou 3 exécutant Windows.

Désormais disponible : Raspberry Pi 3

Passez à la nouvelle carte Raspberry Pi 3 avec Windows 10 IoT Core, totalement prise en charge par le programme d’installation NOOBS. Nous avons publié une nouvelle version de Windows 10 IoT Core Preview dans Windows Insider, afin que vous puissiez profiter pleinement de Raspberry Pi 3.

  • Acheter Raspberry Pi 3
  • Devenir un Windows Insider
  • Prise en main de NOOBS
