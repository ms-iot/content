Tingenes internet (IoT) samler enheder, sensorer, skyen, data og din fantasi.

Hvad er Windows 10 IoT Core?

Opdag de funktioner og den funktionalitet, som Windows 10 IoT Core leverer. Det er forenklet Windows kombineret med IoT-ydeevnen.

Få mere at vide om startpakken

Adafruit har oprettet en startpakke til Windows 10 IoT Core. Pakken indeholder kompatible komponenter, sensorer og dele, som du kan bruge til at komme i gang med at skabe spændende projekter.

Skab din Windows IoT-enhed

Hurtig prototypetestning og udvikling af dine Windows IoT-løsninger på mange forskellige enheder, der kører Windows 10 IoT Core. Med Windows 10 får du effektive værktøjer, der giver dig mulighed for hurtig udvikling og installation på din enhed.

Deltag i vores Maker-community

Få kontakt med andre makers, del kode, og bidrag via GitHub. Deltag i vores community for at se, hvad andre udvikler, og vær med til at påvirke fremtidige versioner af SDK'et.
