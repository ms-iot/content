IoT – internet vašich věcí

Internet věcí (IoT) v sobě spojuje zařízení, snímače, cloud, data a vaši představivost.

Hned začínáme
Podívejte se na novinky: právě teď na Raspberry Pi 3
Co je Windows 10 IoT Core?

Objevte možnosti a funkce, které nabízí Windows 10 IoT Core. Přináší vám jednoduchost Windows v kombinaci s výkonem IoT.

Další informace o balíčku Starter Pack

Společnost Adafruit vytvořila Starter Pack pro Windows 10 IoT Core. Balíček obsahuje kompatibilní komponenty, snímače a součásti, které můžete použít, když začnete vytvářet svoje zajímavé projekty.

Sestavte si zařízení Windows IoT

Začněte si rychle vytvářet prototypy a sestavovat svá řešení Windows IoT na různých zařízeních s Windows 10 IoT Core. Windows 10 vám nabízí účinné nástroje pro rychlý vývoj a rychlé nasazení v zařízení.

Přidejte se ke komunitě tvůrců

Můžete se spojit s dalšími tvůrci, sdílet s nimi kód a publikovat příspěvky přes GitHub. Přidejte se ke komunitě, ať vidíte, co tvoří jiní, a abyste mohli ovlivnit budoucí verze SDK.

Vybrané projekty

Zobrazit všechno
Řízení uzavřené smyčky, dálková čidla a vzdálené uživatelské rozhraní na Raspberry Pi 3
Autor: Windows IoT

Na novém Raspberry Pi 3 budete pomocí PID-regulátoru monitorovat rychlost kola a dosahovat požadovaných otáček.

Světová mapa tvůrců
Autor: Windows IoT

Vytvoříte jednoduchou aplikaci propojenou na internet, která se bude dotazovat na stav webového API a vyžije tyto informace k přepínání LED.

Meteorologická stanice využívající Windows 10
Autor: Windows IoT

Vytvoříte meteorologickou stanici, která využívá nová rozhraní Device API ve Windows 10, díky kterým můžou vývojáři využívat efektivitu Windows a současně řídit vzájemné působení s reálným světem prostřednictvím snímačů a pohánějících zařízení.

Robotická sada
Autor: Windows IoT

Sestavíte robota na kolečkách s využitím robotické sady a potom pomocí desky Raspberry Pi 2 nebo 3 s Windows připojíte USB joystick pro ovládání směru tohoto zařízení.

Raspberry Pi 3 je už dostupné

Ošahejte si novou desku Raspberry Pi 3 s Windows 10 IoT Core, která je teď součástí instalačního programu NOOBS. Přes program Windows Insider jsme vydali novou verzi Insider Preview produktu Windows 10 IoT Core, s kterou budete moct plně využít výhod Raspberry Pi 3.

  • Koupit Raspberry Pi3
  • Staňte se účastníkem programu Windows Insider
  • Začínáme s NOOBS
