Das Internet „Ihrer“ Dinge

Das Internet der Dinge (Internet of Things, IoT) vereint Geräte, Sensoren, Cloud, Daten und Ihre Ideen.

Was ist Windows 10 IoT Core?

Entdecken Sie die Features und Funktionen von Windows 10 IoT Core. Die Benutzerfreundlichkeit von Windows vereint mit der Leistungsstärke von IoT

Weitere Informationen zum Starter Pack

Adafruit hat ein Starter Pack für Windows 10 IoT Core erstellt. Das Pack enthält kompatible Komponenten, Sensoren und Teile, die Sie zum Einstieg in die Erstellung aufregender Projekte verwenden können.

Entwickeln Ihres Windows IoT-Geräts

Erstellen Sie in kurzer Zeit Windows IoT-Prototypen und -Lösungen für eine Vielzahl von Geräten unter Windows 10 IoT Core. Die leistungsfähigen Tools in Windows 10 sorgen für eine schnelle Entwicklung und Bereitstellung auf Ihren Geräten.

Beitreten zur Maker-Community

Vernetzen Sie sich mit anderen Herstellern, um Code auszutauschen und per GitHub einen Beitrag zu leisten. Treten Sie der Community bei, um die Produkte anderer Benutzer zu sehen und Einfluss auf zukünftige SDK-Versionen zu nehmen.
