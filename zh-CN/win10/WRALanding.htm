Windows Remote Arduino

准备好使用 Windows Remote Arduino 进行开发

概述

我们的产品/服务有两个主要组件：Windows Remote Arduino 库（在 GitHub 上称为“远程连接”，链接在此处）和 Windows Remote Arduino 体验应用。 Windows Remote Arduino 库是一种开源 WinRT 库，可将任何 Windows 10 设备（可能是 Windows Phone、Surface Pro、Windows 电脑，甚至是 Raspberry Pi 2 或 3）连接到运行 StandardFirmata 的 Arduino 开发板。 通过编写在 Windows 10 设备上使用库的程序，开发人员可以在 Arduino 开发板上远程控制每个引脚。制造商可以通过蓝牙、USB、WLAN 或以太网控制他们的 Arduino，并且他们可以使用任何 WinRT 语言（C++/CX、C# 和 Javascript）执行此操作。 Windows Remote Arduino 体验应用是为演示 Windows Remote Arduino 库的强大功能而生成的 UI 体验。 你可以在任何 Windows 10 设备上运行该应用，并且在运行 StandardFirmata 的 Arduino 开发板上使用该应用控制引脚。 此应用可以完美试验 Windows Remote Arduino 库的功能！

开发和浏览

如果你想使用 Windows Remote Arduino 库生成你的第一个应用，请遵循此链接以在 Arduino 上远程控制 LED。

如果你想在不深入研究代码的情况下查看该技术在执行操作时的运行方式，请遵循此链接以开始使用 Windows Remote Arduino 体验应用。

功能

Windows Remote Arduino 库支持以下功能：
  1. GPIO - 模拟和数字 I/O
    • 数字写入
    • 数字读取
    • 模拟写入
    • 模拟读取
    • 为任意引脚设置引脚模式
    • 接收值发生更改或进行报告时的事件。
  2. I2C/TwoWire
    • 通过 I2C 将数据发送到设备或从中接收数据。
  3. 通过 Firmata SysEx 命令自定义协议
