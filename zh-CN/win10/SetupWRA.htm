所需内容

在本部分中，我们准备一台 Arduino 设备，通过之前准备的 Windows 10 设备远程控制它。 Arduino 需要运行 Firmata 版本，从而允许开发板通过 USB、蓝牙或 WLAN 与应用进行通信。 你可以决定所需的通信连接 - 请务必相应地选择以下部分：

你将需要：

  1. 一台 Windows 设备 - 不一定是上一步骤中准备好的设备，可用来将草图上载到 Arduino 的任何设备皆可。
  2. 一块 Arduino 开发板 - 我们的示例已经过测试，可适用于 Arduino Uno。
  3. 标准 A 到标准 B USB 电缆 - 无论选择的是何种连接，只要是上载 Arduino 草图，都需要此电缆。
  4. 适用于 Arduino 的蓝牙设备（可选）- 仅当你选择通过蓝牙连接时才需要。我们建议使用 SparkFun Bluetooth Mate Silver。
  5. 适用于 Arduino 的 WLAN 设备（可选）- 仅当你选择通过 WLAN 连接时才需要。我们建议使用此 Arduino WLAN Shield。

Arduino 设置

Windows Remote Arduino 使用 Firmata 协议，该协议已在多种语言版本中实现，包括 Arduino。Arduino 实现称为 StandardFirmata，并且在安装时通过与 Arduino 软件一起预打包来提供。请按照下面的步骤将 StandardFirmata 草图上载到你的 Arduino。

  1. 从 http://arduino.cc 下载并安装 Arduino 软件。
  2. 使用 USB 将你的 Arduino 设备连接到计算机。
  3. 启动 Arduino 应用程序。
  4. 请验证你在“工具”>“开发板”下选择了正确的 Arduino 开发板。
  5. 请验证你在“工具”>“端口”下选择了正确的 COM 端口。 每个选项的旁边都应该显示开发板的名称，以便可以更轻松地选择正确的选项。
  6. 在 Arduino IDE 中，导航到“文件”>“示例”>“Firmata”>“StandardFirmata”。
  7. 按“上载”以将 StandardFirmata 草图部署到 Arduino 设备。

除非使用不同的草图重新编程，否则你的 Arduino 从现在开始将永远运行 StandardFirmata 草图。你现在可以选择将 Arduino 从计算机中断开连接，然后采用任何所选方式接通电源。

此时，可以使用 USB 连接（无需进行其他操作）- 如果你选择此选项，请转到下一个页面。 否则，请按照以下指令集之一设置蓝牙或 WLAN 设备。

如果你想要在设备之间使用推荐的蓝牙配对，则需要将蓝牙设备连接到 Arduino。 按照我们 GitHub 上的步骤设置蓝牙模块，然后再转到下一步。

如果你想要在设备之间使用 WLAN 配对，则需要将 WLAN 设备连接到 Arduino。 按照我们 GitHub 上的步骤设置蓝牙模块，然后再转到下一步。
