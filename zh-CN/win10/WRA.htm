Windows Remote Arduino

准备好计算机以针对 Windows Remote Arduino 和 Windows 10 IoT 核心版进行开发，然后生成你的第一个 Windows IoT 核心版应用。

Windows Remote Arduino 是一个开源 Windows 运行时组件库，可允许制造商通过蓝牙、USB、WLAN 或以太网连接控制 Arduino！ 它适用于希望使用 Windows 运行时语言控制 Arduino 硬件强大功能的 Windows 运行时 (WinRT) 开发人员。在其项目中包含此组件的开发人员将自动获得使用任何 WinRT 语言（C++/CX、C# 和 JavaScript）访问其功能的权限。库在 Firmata 协议的基础之上进行生成，该协议已在预打包的 Firmata 库（包含在从 http://arduino.cc 获取的 Arduino 软件中）中实现。

概述

Windows Remote Arduino 桥接软件世界和真实世界之间的间隙。通过利用 Windows 10 的强大功能，我们可以展现 Windows 设备（如 Windows Phone、Surface Pro 3、Windows 台式机甚至是 Raspberry Pi 2）的新功能。将设备与 Arduino 进行配对，举手之间即可进入一个全新的世界。

功能

Windows Remote Arduino 启用了以下功能，即“开箱即用”。
  1. GPIO - 模拟和数字 I/O
    • 数字写入
    • 数字读取
    • 模拟写入
    • 模拟读取
    • 为任意引脚设置引脚模式
    • 接收值发生更改或进行报告时的事件。
  2. I2C/TwoWire
    • 通过 I2C 将数据发送到设备或从中接收数据。
  3. 通过 Firmata SysEx 命令自定义协议
