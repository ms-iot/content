第 1 课： 简介和设置

{% include kit-steps.html device=page.deviceName %}

Windows 10 IoT 核心版是新版本的 Windows，面向小型嵌入式设备和制造商开发板，例如 Raspberry Pi2。 IoT 核心版旨在适用于低级别的总线接口，如 I2C、SPI、USB。可以使用 Visual Studio 中的多种编程语言编写通用 Windows 应用程序来快速创建制造商项目。

Windows 10 IoT 核心版可用于读取传感器数据、控制传动装置、连接到云、创建 IoT 应用程序等。Adafruit 中的初学者包包括入门部分。

查看包含在工具包中的部分、设置电脑和编程环境、设置 Raspberry Pi 2 开发板，然后你可以随时创建你的第一个项目。

让我们开始吧！
