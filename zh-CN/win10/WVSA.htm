设置 Windows Virtual Shields for Arduino

使 Arduino 和 Windows 10 设备准备好与 Windows Virtual Shields for Arduino 一起使用。

概述

如果你已使用 Arduino，则已经熟悉防火墙的概念。每个防火墙都有特殊的用途（例如温度防火墙、加速计防火墙），使用多个防火墙生成设备将非常复杂、成本高且空间效率低。现在，假设你可以使用低成本的 Windows Lumia 手机作为一组压缩的防火墙。你的 Arduino 草图可通过易于使用的库调用来访问 Windows Phone 中价值数百美元的传感器和功能。 这正是 Windows Virtual Shields for Arduino 库为开发人员启用的功能。还有一个更好的优势，即此技术适用于所有 Windows 10 设备，所以你也可以在电脑或 Surface 上使用这些传感器和功能。此外，Arduino 可以卸载 Windows 10 配套设备上耗费计算的任务（例如语音识别和 Web 解析）！ 让我们了解如何设置硬件和软件以与 Windows Virtual Shields for Arduino 一起使用。

1.使用 Windows Virtual Shields for Arduino 让你的 Windows 10 设备准备好开发项目。隐藏

在教程的这一部分中，你将通过使用 Windows Virtual Shields for Arduino 应用程序（该通用 Windows 应用程序允许你使用你的设备作为 Arduino 开发板的“虚拟防火墙”）加载来准备好你选择的 Windows 10 设备。 这将为制造商带来一些强大的功能，从而使他们能够相对轻松地使用语音识别、触摸屏以及 Windows 的计算功能。

硬件

Windows Virtual Shields for Arduino 应用程序可以在任何 Windows 10 设备上运行，但本教程将介绍如何设置 Windows Phone。

所需内容

运行 Windows 10 的 Windows Phone - 我们建议使用 Lumia 520 或 Lumia 635。

设置 Windows Phone

如果你的手机尚未运行 Windows 10，可以选择安装该软件的预览版。 Windows Phone 8 用户可以转到 Microsoft 官方商城应用下载“Windows 预览体验成员”应用程序 - 此应用允许用户选择以更新方式接收 Windows 10 Technical Preview。 打开该应用后，按照提示和说明进行操作，直到你的手机成功运行 Windows 10 为止。

软件

有两个选项可用于在 Windows Phone 上安装 Windows Virtual Shields for Arduino UWP 应用。 比较简单快捷的方法是下载该应用。

  1. 从 Microsoft 官方商城下载该应用
  2. 使用电脑和 Visual Studio 旁加载该应用

选项 1： 从 Microsoft 官方商城下载该应用

遵循此 Windows Virtual Shields for Arduino 的 Microsoft 官方商城页面的链接来下载应用程序，然后进行安装。然后，打开应用程序以确保它可以正常运行。 你的设备现已设置为用作 Arduino 的“虚拟防火墙”！ 你可以继续到下一个页面。

选项 2： 使用电脑和 Visual Studio 旁加载该应用

所需内容

  1. Visual Studio 2015，用于将 Windows Virtual Shields for Arduino 应用旁加载到开发人员解锁的手机上。
  2. 此存储库，包含 Windows Virtual Shields for Arduino 应用程序的代码。 克隆此存储库或将其作为 ZIP 下载到本地磁盘上。 如果你不熟悉 Git 并想要进行正确的克隆，请按照此处的说明进行操作。

设置 Visual Studio 2015

  1. 使用 dev.windows.com 中的 Windows 10 开发者预览版工具安装 Visual Studio 2015。 我们建议使用 Visual Studio 免费社区版，但也可以使用企业版和专业版。 如果已安装 Visual Studio，请跳到下一步。
  2. 在 Visual Studio 中，从上面的“所需内容”部分中下载的存储库中加载 Shield.sln。
  3. 确保 Windows 10 设备（在此情况下是 Windows Phone）已经过开发人员解锁。（此页面介绍如何解锁 Windows Phone 8.1、8 和 7.1；但是，对于 Windows 10 手机，步骤完全相同）。
  4. 将 Windows 10 设备插入计算机，并将 Shield.sln 程序部署到设备。 若要执行此操作，请先部署到“本地计算机”，并确保将设备的体系结构设置为“ARM”。
  5. 在手机上运行新安装的 Windows Virtual Shields for Arduino 应用程序以确保部署已成功。 你的 Windows 10 设备现已设置为用作虚拟防火墙！
