设置 Windows Virtual Shields for Arduino

为适用于 IoT 的 Windows 开发人员计划准备好计算机、设置 Raspberry Pi 2，然后生成你的第一个 Windows IoT 核心版应用。

1.准备好电脑以使用 Virtual Shields for Arduino 开发 IoT 应用程序。隐藏

在教程的这一部分中，你将通过使用 Virtual Shields for Arduino 应用程序（该程序允许你使用所选设备作为 Arduino 开发板的虚拟防火墙）加载准备好你选择的 Windows 10 设备。 这将为制造商带来一些强大的功能，从而使他们能够相对轻松地使用语音识别、触摸屏幕以及 Windows 设备的其他资源。

硬件

Virtual Shields for Arduino 应用程序可以在任何 Windows 10 设备上运行，但本特定示例将使用 Windows Phone。

所需内容

运行 Windows 10 的 Windows Phone - 我们建议使用 Lumia 520 或 Lumia 635。

设置 Windows Phone

如果你的手机尚未运行 Windows 10，可以选择安装该软件的预览版。 Windows Phone 8 用户可以转到 Microsoft 官方商城应用下载“Windows 预览体验成员”应用程序 - 此应用允许用户选择以更新方式接收 Windows 10 Technical Preview。 打开该应用后，按照提示和说明进行操作，直到你的手机成功运行 Windows 10 为止。

软件

在 Windows Phone 上安装 Virtual Shields for Arduino 的选项有两个。

  1. 从 Microsoft 官方商城下载该应用。
  2. 使用电脑和 Visual Studio 旁加载该应用。

比较方便快捷的方法是下载该应用。

选项 1： 从 Microsoft 官方商城下载该应用

按照此链接进入该应用的 Microsoft 官方商城页面、下载该文件，随后进行安装。然后，打开应用程序以确保它可以正常运行。

你的设备现已设置为用作 Arduino 的虚拟防火墙！ 你可以继续到下一个页面。

选项 2： 使用电脑和 Visual Studio 旁加载该应用

所需内容

  1. Visual Studio 2015，用于将 Virtual Shields for Arduino 应用旁加载到开发人员解锁的手机上。
  2. 此存储库，包含 Virtual Shields for Arduino 应用程序的代码。 克隆此存储库或将其作为 ZIP 下载到本地磁盘上。 如果你不熟悉 Git 并想要进行正确的克隆，请按照此处的说明进行操作。

设置 Visual Studio 2015

  1. 使用 dev.windows.com 中的 Windows 10 开发者预览版工具安装 Visual Studio 2015。 我们建议使用 Visual Studio 社区版，但也可以使用企业版和专业版。 如果已安装 Visual Studio，请跳到下一步。
  2. 在 Visual Studio 中，从上面的“所需内容”部分中下载的存储库中加载 Shield.sln。
  3. 确保你的手机已经过开发人员解锁（此页面介绍如何解锁 Windows Phone 8.1、8 和 7.1；但是，对于 Windows 10 手机，步骤完全相同）。
  4. 将 Shield.sln 程序部署到设备。 若要执行此操作，请先部署到本地计算机，并确保将设备的体系结构设置为“ARM”。
  5. 在手机上运行 Virtual Shields for Arduino 应用程序以确保部署已成功。 你的 Windows 10 设备现已设置为用作虚拟防火墙！
