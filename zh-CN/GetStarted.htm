设置 Windows 10 IoT 核心版设备

如果你有一个 Raspberry Pi 2 或 MinnowBoard MAX，可以使用 Windows 10 IoT 核心版仪表板轻松地对其进行设置和配置。可以使用该仪表板在 RTM（公共）版本的 Windows 10 IoT 核心版上进行设置，并且需要运行 Windows 10 的电脑。

请确保运行的是最新公共版本的 Windows 10 或 Windows 10 Insider Preview。如果不是，可从此处升级。

Arduino 和 Windows 10 IoT 核心版

Windows 10 和 Arduino 共同协作以帮助你创建令人惊叹的项目，单击以下链接之一即可开始。

Arduino 接线和 UWP Lightning 提供程序

为了支持在 IoT 核心版设备上使用现有的 Arduino 接线草图，现在可以在 Visual Studio 中使用 Arduino 接线的项目模板。这些项目使用可提供高性能的直接内存映射的驱动程序。此外，基于相同的驱动程序，可以在 UWP 应用中使用一组控制器提供程序来访问可提供高性能的 GPIO、I2C 和 SPI 控制器。

Windows Remote Arduino

Windows Remote Arduino 是一个开源 Windows 运行时组件库，可允许制造商通过蓝牙或 USB 连接控制 Arduino。它适用于希望使用 Windows 运行时语言控制 Arduino 硬件电源的 Windows 运行时开发人员。在其项目中包含此组件的开发人员将自动获得使用任何 WinRT 语言（C++/CX、C# 和 JavaScript）访问其功能的权限。

Windows Virtual Shields for Arduino

Windows Virtual Shields for Arduino 是一个开源库，主要用于与在所有 Window 10 设备（包括 Windows Lumia 手机）上运行的开源通用 Windows 应用程序通信的 Arduino UNO。该库向 Arduino 接线草图公开 Lumia 手机的传感器和功能。
