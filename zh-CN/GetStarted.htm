Windows 10 IoT 核心版的远程屏幕体验

使用此新技术将屏幕放在任何无屏幕的设备上。Windows 10 IoT 核心版的最新预览体验成员版本支持这一新的远程屏幕体验，使用户可以从任何 Windows 10 台式电脑、平板电脑或手机查看和控制他们的 IoT 核心版 UWP 应用程序。

该技术还使用户可以使用来自他们的任何 Windows 10 设备的传感器数据控制他们的 IoT 核心版应用程序。 此远程传感器功能允许你将你的 IoT 项目扩展到整个 Windows 10 设备的生态系统。

此功能是自带功能，并且易于入门。 了解如何远程处理 Windows 10 IoT 核心版设备。


Arduino 和 Windows 10

Arduino 和 Windows 10 共同协作以帮助你创建令人惊叹的项目，单击以下链接之一即可开始。我们的技术支持 Arduino Uno 和 Mega 以及新的 Arduino 101。

Arduino 接线和 UWP Lightning 提供程序

为了支持在 IoT 核心版设备上使用现有的 Arduino 接线草图，现在可以在 Visual Studio 中使用 Arduino 接线的项目模板。这些项目使用可提供高性能的直接内存映射的驱动程序。此外，基于相同的驱动程序，可以在 UWP 应用中使用一组控制器提供程序来访问可提供高性能的 GPIO、I2C 和 SPI 控制器。

开始使用 Arduino 接线

Windows Remote Arduino

Windows Remote Arduino 是一个开源 Windows 运行时库，可允许制造商通过蓝牙、USB、WLAN 或以太网连接控制 Arduino。它允许开发人员在其 Windows 项目中包含 Arduino 传感器，甚至将计算成本昂贵的工作从 Arduino 卸载到 Windows。Windows Remote Arduino 库可与任何 WinRT 语言（C++/CX、C# 和 JavaScript）结合使用。

Windows Virtual Shields for Arduino

Windows Virtual Shields for Arduino 是一个开源库，主要用于与在所有 Window 10 设备（包括 Windows Lumia 手机）上运行的开源通用 Windows 应用程序通信的 Arduino UNO。该库向 Arduino 接线草图公开 Lumia 手机的传感器和功能。
