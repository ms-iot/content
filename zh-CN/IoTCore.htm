了解 Windows 10 IoT 核心版

Windows 10 IoT 核心版是 Windows 10 的一个版本，针对有屏幕或无屏幕的小型设备进行了优化，并且可在 Raspberry Pi 2、Arrow DragonBoard 410c 和 MinnowBoard MAX 上运行。Windows 10 IoT 核心版利用丰富可扩展的通用 Windows 平台 (UWP) API 来生成出色的解决方案。 

为什么使用 Windows 10 IoT 核心版？

Windows 10 IoT 核心版会将 Windows 的强大功能引入你的设备，并可轻松将更丰富的体验与你的设备集成，如自然用户界面、搜索、网盘和基于云的服务。


通用 Windows 平台和硬件

UWP API 使你可以编写一个应用程序并在设备、手机或台式机上使用它，同时使你能够访问数千个要在项目中使用的 Windows 支持的设备。

Arduino 接线 API

Windows 10 IoT 核心版还支持在 Arduino 草图和库中使用的易用 Arduino 接线 API，以实现直接硬件访问。

Visual Studio

若要开发应用程序，可以使用最新的免费 Visual Studio Community 版。Visual Studio 是一款专业质量开发工具，可免费提供，并包含了通用应用模板、代码编辑器、强大的调试程序以及丰富的语言支持等。

连接点

利用 Connect-the-Dots 等开放框架的强大功能帮助你将设备连接到 Microsoft Azure。借助 Microsoft Azure，你可以利用高级分析服务实现出色的解决方案。

是否适用于我？

制造商

通对 Arduino 接线 API 和 Windows Remote Arduino 的支持，并且可用于常见的制造商开发板，Windows 10 IoT 核心版对所有制造商项目而言都是一个不错的选择。使用 Windows 的强大功能可以在你最喜爱的设备上生成无数个创新项目。

商业组装商

对于商业组装商，Windows 10 IoT 核心版提供了一个出色的平台来生成占用较少且低成本的行业设备，其中具有与 Windows 10 的其他版本相似的企业级安全、管理和服务。

硬件要求

Windows 10 IoT 核心版的要求取决于是在有外设模式下还是在无外设模式下运行。有外设设备具有视频屏幕并使用 Windows 视频子系统来处理它。无外设设备没有屏幕。

内存

无外设

256 MB RAM（操作系统免费有 128 MB）/ 2 GB 存储

有外设

512 MB RAM（操作系统免费有 256 MB）/ 2 GB 存储

处理器

400 MHz 或更快
（x86 需要 PAE、NX 和 SSE2 支持）
