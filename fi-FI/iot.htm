Mikä on Windows 10 IoT Core?

Tutustu Windows 10 IoT Coren ominaisuuksiin ja toimintoihin. Se yhdistää Windowsin helppokäyttöisyyden IoT:n tehokkuuteen.

Lisätietoja aloittelijan paketista

Adafruit on luonut Windows 10 IoT Coren aloittelijan paketin. Paketti sisältää yhteensopivia komponentteja, tunnistimia ja osia, joilla voit aloittaa mielenkiintoisten projektien kehittämisen.

Kehitä Windows IoT -laite

Luo prototyyppi ja kehitä Windows IoT -ratkaisuja nopeasti erilaisiin Windows 10 IoT Core -laitteisiin. Windows 10:n tehokkaat työkalut nopeuttavat kehitystyötä ja helpottavat sovelluksen käyttöönottoa laitteessa.

Liity tekijöiden yhteisöön

Pidä yhteyttä muihin tekijöihin, jaa koodeja ja osallistu GitHubin toimintaan. Liittymällä yhteisöön näet, mitä muut kehittävät ja mikä on muiden panos SDK:n tulevissa julkaisuissa.

Raspberry Pi 3 nyt saatavana

Hanki uusi Windows 10 IoT Core -pohjainen, NOOBS-asennusohjelman tukema Raspberry Pi 3 -emolevy. Olemme julkaisseet Windows Insider -ohjelman kautta uuden Windows 10 IoT Core Insider Preview -version, jonka avulla voit hyödyntää Raspberry Pi 3:a parhaalla mahdollisella tavalla.
