O que é o Windows 10 IoT Core?

Descubra as funcionalidades que o Windows 10 IoT Core oferece. É a simplicidade do Windows combinada com a potência da IoT.

Obter mais informações acerca do Pacote de Introdução

A Adafruit criou um Pacote de Introdução para o Windows 10 IoT Core. O pacote inclui componentes compatíveis, sensores e partes que pode utilizar para começar a criar projetos incríveis.

Crie o seu dispositivo Windows IoT

Crie rapidamente protótipos e soluções Windows IoT em vários dispositivos que executem o Windows 10 IoT Core. O Windows 10 oferece-lhe poderosas ferramentas que lhe permitem programar rapidamente e fazer a implementação no seu dispositivo.

Junte-se à Comunidade de criadores

Ligue-se a outros criadores para partilhar código e contribuir através do GitHub. Junte-se à comunidade para ver o que outras pessoas estão a criar e influencie as versões futuras do SDK.
