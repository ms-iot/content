Η δική σας τεχνολογία Internet of things

Η τεχνολογία Internet of Things (IoT) συγκεντρώνει συσκευές, αισθητήρες, cloud, δεδομένα και τη φαντασία σας.

Ξεκινήστε τώρα
Δείτε τι νέο υπάρχει: Τώρα σε Raspberry Pi 3
Τι είναι τα Windows 10 IoT Core;

Ανακαλύψτε τις δυνατότητες και λειτουργίες που προσφέρουν τα Windows 10 IoT Core. Η ευκολία χρήσης των Windows συνδυάζεται με την ισχύ του IoT.

Ενημερωθείτε για το πακέτο εκκίνησης

Η Adafruit έχει δημιουργήσει ένα πακέτο εκκίνησης για τα Windows 10 IoT Core. Το πακέτο περιλαμβάνει συμβατά στοιχεία, αισθητήρες και εξαρτήματα που μπορείτε να χρησιμοποιήσετε, για να αρχίσετε να δημιουργείτε συναρπαστικά έργα.

Δημιουργία της δικής σας συσκευής Windows IoT

Δημιουργήστε γρήγορα τα δικά σας πρωτότυπα και λύσεις Windows IoT σε μια σειρά από συσκευές με Windows 10 IoT Core. Τα Windows 10 σάς παρέχουν ισχυρά εργαλεία που σας εξασφαλίζουν γρήγορο προγραμματισμό και ανάπτυξη στη συσκευή σας.

Συμμετοχή στην κοινότητα δημιουργών

Συνδεθείτε με άλλους δημιουργούς, για να μοιραστείτε κώδικα και να παρουσιάσετε τη συνεισφορά σας μέσω του GitHub. Συμμετάσχετε στην κοινότητα, για να δείτε τις δημιουργίες άλλων ατόμων και να επηρεάσετε τις μελλοντικές εκδόσεις του SDK.

Επιλεγμένα έργα

Προβολή όλων
Έλεγχος κλειστού βρόγχου, απομακρυσμένοι αισθητήρες και απομακρυσμένο UX σε Raspberry Pi 3
Από Windows IoT

Χρησιμοποιήστε έναν ελεγκτή PID για να παρακολουθείτε την ταχύτητα του τροχού και να φτάνετε στα RPM που θέλετε στο νέο Raspberry Pi 3.

Παγκόσμιος χάρτης δημιουργών
Από τα Windows IoT

Δημιουργήστε μια απλή εφαρμογή με σύνδεση στο Internet, για να αποστέλλετε ερωτήματα σχετικά με την κατάσταση ενός API web και να χρησιμοποιείτε τις συγκεκριμένες πληροφορίες για την ενεργοποίηση ή απενεργοποίηση μιας ενδεικτικής λυχνίας.

Μετεωρολογικός σταθμός με Windows 10
Από τα Windows IoT

Δημιουργήστε έναν μετεωρολογικό σταθμό που χρησιμοποιεί τα νέα API συσκευών στα Windows 10, για να αξιοποιήσει την ισχύ των Windows, αλληλεπιδρώντας ταυτόχρονα με τον πραγματικό κόσμο μέσω αισθητήρων και ενεργοποιητών.

Κιτ robot
Από Windows IoT

Δημιουργήστε ένα τροχήλατο robot χρησιμοποιώντας ένα κιτ robot και, στη συνέχεια, συνδέστε ένα joystick USB για να ελέγξετε την κίνησή του, χρησιμοποιώντας την πλακέτα Raspberry Pi 2 ή 3 με Windows.

Το Raspberry Pi 3 είναι τώρα διαθέσιμο

Αποκτήστε τη νέα πλακέτα Raspberry Pi 3 με Windows 10 IoT Core, τα οποία αποτελούν πλέον μέρος του προγράμματος εγκατάστασης NOOBS. Κυκλοφορήσαμε μια νέα έκδοση Windows 10 IoT Core Insider Preview μέσω του προγράμματος Windows Insider, η οποία θα σας επιτρέψει να αξιοποιήσετε πλήρως το Raspberry Pi 3.
