사물 인터넷

IoT(사물 인터넷)를 통해 장치, 센서, 클라우드, 데이터 및 상상을 통합할 수 있습니다.

지금 바로 시작
새로운 기능 보기: Raspberry Pi 3의 새로운 기능
Windows 10 IoT Core란?

Windows 10 IoT Core에서 제공하는 기능을 살펴보세요. Windows 10 IoT Core에는 Windows의 편의성과 IoT의 강력한 기능이 결합되어 있습니다.

스타터 팩 자세히 알아보기

Adafruit에서 Windows 10 IoT Core를 위한 스타터 팩을 만들었습니다. 이 팩에는 호환 구성 요소, 센서, 파트가 포함되어 있어 흥미로운 프로젝트 시작에 활용할 수 있습니다.

Windows IoT 장치 구축

Windows 10 IoT Core를 실행하는 다양한 장치에서 Windows IoT 솔루션을 신속하게 프로토타입하고 빌드할 수 있습니다. Windows 10은 빠르게 개발하여 장치에 배포할 수 있게 해주는 강력한 도구를 제공합니다.

제작자 커뮤니티 참여

다른 제작자와 연결하여 코드를 공유하고 GitHub를 통해 기여합니다. 커뮤니티에 참여하여 다른 사람들은 어떤 프로젝트를 만드는지, 그들이 미래의 SDK 릴리스에 어떤 영향을 주는지 알아보세요.

추천 프로젝트

모두 보기
Raspberry Pi 3에서의 폐쇄형 루프 제어, 원격 센서 및 원격 UX
작성자: Windows IoT

PID 컨트롤러를 사용하여 휠 속도를 모니터링하고 새로운 Raspberry Pi 3에서 원하는 RPM을 꾸준히 유지합니다.

제작자 세계 지도
작성자: Windows IoT

간단한 인터넷 연결 응용 프로그램을 만들어서 웹 API 상태를 쿼리하고 이 정보를 이용하여 LED를 켜고 끕니다.

Windows 10을 사용하는 기상 관측 스테이션
작성자: Windows IoT

Windows 10의 새로운 디바이스 API를 통해 Windows의 강력한 기능을 활용하는 한편 센서 및 작동기를 통해 실제 세상과 상호 작용하는 기상 관측 스테이션을 만듭니다.

로봇 키트
작성자: Windows IoT

바퀴 달린 로봇을 만들어서 USB 조이스틱을 연결하면 Windows를 실행하는 Raspberry Pi 2 또는 3 보드를 사용하여 방향을 제어할 수 있습니다.

Raspberry Pi 3 지금 사용 가능

이제 NOOBS 설치 프로그램의 일부인 Windows 10 IoT Core가 탑재된 새로운 Raspberry Pi 3 보드를 만나보세요. Raspberry Pi 3의 모든 기능을 활용할 수 있도록 Windows 참가자를 통해 새로운 Windows 10 IoT Core Insider Preview를 공개했습니다.
