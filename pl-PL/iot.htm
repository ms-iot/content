Co to jest system Windows 10 IoT Core?

Odkryj funkcje i możliwości udostępniane przez system Windows 10 IoT Core. Prostota systemu Windows i potencjał technologii IoT.

Więcej informacji o pakiecie startowym

Firma Adafruit utworzyła pakiet startowy dla systemu Windows 10 IoT Core. Pakiet obejmuje zgodne składniki, czujniki i części, których można użyć podczas tworzenia ciekawych projektów.

Utwórz własne urządzenie z systemem Windows IoT

Szybko twórz prototypy i kompletne rozwiązania dla systemu Windows IoT na różnych urządzeniach z systemem Windows 10 IoT Core. System Windows 10 oferuje zaawansowane narzędzia umożliwiające szybkie tworzenie i wdrażanie aplikacji na urządzeniu.

Dołącz do społeczności makerów

Nawiąż kontakt z innymi makerami, aby udostępniać kod i wprowadzać modyfikacje za pośrednictwem witryny GitHub. Dołącz do społeczności, aby zobaczyć, nad czym pracują inni, i wpłynąć na kształt przyszłych zestawów SDK.
