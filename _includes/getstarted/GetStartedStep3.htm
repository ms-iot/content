Step 3 of 4: Set up Visual Studio

Set up your development environment to explore and run the IoT samples.

Set up your developement environmentHide

  1. Download and install Visual Studio Community 2017 from here

    NOTE: Visual Studio Professional 2017 and Visual Studio Enterprise 2017 can be downloaded from here. If you choose to install a different edition of Visual Studio make sure to do a custom installation and select Universal Windows App Development Tools > Tools and Windows SDK.

  2. Install the Windows IoT project templates

    Download and install the IoT templates from the Visual Studio Gallery here.
    Alternatively, the templates can be found by searching for Windows IoT Core Project Templates in the Visual Studio Gallery or directly from Visual Studio in the Extension and Updates dialog (Tools > Extensions and Updates > Online).

  3. Validate your installation

    Select Help > About Microsoft Visual Studio to display version information.

    The required version of Visual Studio is 15.1 (26403.0) Release.

    The required version of Visual Studio Tools for Universal Windows Apps is 15.0.26403.00.

  4. Enable developer mode on your device running Visual Studio. Instructions are here.

Step 3 of 4: Set up Visual Studio is complete. Write your first app is next.
