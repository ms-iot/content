Step 2 of 4: Set up your device

Use NOOBS to install Windows 10 IoT Core.

1Setup your device and connect it to the network.Hide

  1. Insert your NOOBS-ready micro SD Card into your Raspberry Pi. Ensure a monitor, keyboard, and mouse are also connected.
  2. If you are able to, connect your Raspberry Pi to the Internet by Ethernet cable. Boot your Raspberry Pi.
  3. If you haven't yet connected the device to a network follow the on-screen instructions to connect to your WiFi network.

2Follow the instructions to install Windows 10 IoT CoreHide

  1. Select Windows 10 IoT Core from the list of available options, and then click install.
  2. NOOBS will ask to confirm you wish to overwrite the contents of the SD Card before installing Windows 10 IoT Core. Unless you have data on the card you wish to save, press yes. Once the installation process starts, it cannot be undone.
  3. Select the RTM (official release) of Windows 10 IoT Core, or the latest Windows 10 IoT Core Insider build to install onto your SD Card. Finally, view and accept the license agreement.
  4. Optional. If you have selected the Windows 10 IoT Core Insider build, you must log into a valid Microsot Account (MSA) Insider account. If you don't have one, you can sign up within NOOBS, or at the Windows Insider website. Being a Windows Insider lets you try out new versions of Windows before everyone else, giving you early access to new features and capabilities before everyone else.
  5. Windows 10 IoT Core is now installing. After a few minutes the installation should complete. Pressing "OK" will cause your device to reboot into Windows 10 IoT Core!

If using WiFi, you'll need to re-configure your WiFi settings after booting into Windows 10 IoT Core. You can either use the IoT Core Dashboard or the Windows Device Portal to do this.

3Try out the Quick-Run samples using the IoT Core DashboardHide

You can download and install the IoT Core Dashboard to interact with and manage your device. By downloading and using the Windows 10 IoT Core Dashboard you agree to the license terms and privacy statement for Windows 10 IoT Core Dashboard.
