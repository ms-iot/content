The Joule has onboard WiFi that you may have already configured during the initial bootup of the default app. You can use the steps below to change wi-fi settings. If you prefer using wired Ethernet, you can connect using a USB Ethernet dongle.

Connecting to a Wireless Network

  1. Click the Settings button next to the clock on the Win10 IoT Core default application.
  2. On the Settings page, select "Network & Wi-Fi" via the menu on the left.
  3. Your Joule will automatically begin scanning for wireless networks.
  4. Once your network appears in the list, select it and click the "Connect" button to connect.
